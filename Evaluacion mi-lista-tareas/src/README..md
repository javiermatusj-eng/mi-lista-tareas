# 📋 Aplicación de Lista de Tareas con React

## ¿Qué hace esta app?
Permite agregar y eliminar tareas de forma interactiva. Es una aplicación web creada con **React** y **Vite** que demuestra el uso de:
- Componentes funcionales
- Estado con `useState`
- Manejo de eventos (formulario, clics)
- Renderizado dinámico de listas

## Tecnologías
- React 18
- Vite
- CSS puro (sin frameworks externos)
